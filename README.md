# Repositorio de POO 

Este repositorio contiene los ejercicios, ejemplos y prácticas desarrollados durante las clases de Programación Orientada a Objetos.

## Objetivo
El objetivo de este repositorio es almacenar las soluciones vistas en clase y servir como material de apoyo para repasar los conceptos aprendidos y practicar programación.

## Contenido
En este repositorio se pueden encontrar:

- Ejercicios resueltos en clase
- Otros ejemplos

## Uso
Los archivos pueden utilizarse como referencia para estudiar, practicar y comprender mejor los conceptos explicados durante las clases.
