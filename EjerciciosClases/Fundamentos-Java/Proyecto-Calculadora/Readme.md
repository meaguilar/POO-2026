# 🧮 Calculadora Básica en Java

## 📌 Descripción del Proyecto

Este ejercicio tiene como objetivo la creación de una calculadora básica que realice las cuatro operaciones aritméticas fundamentales:

- Suma  
- Resta  
- Multiplicación  
- División  

El propósito principal es que los estudiantes practiquen y refuercen:

- Sintaxis básica de Java  
- Uso del modificador `static` en funciones  
- Creación y uso de funciones para cada operación  
- Control de flujo (`if`, `switch`, `do-while`)  
- Entrada y salida de datos por consola  

---

## 🎯 Funcionalidades del Proyecto

El sistema debe permitir:

- Ingresar números desde consola  
- Seleccionar la operación a realizar  
- Ejecutar la operación mediante funciones estáticas  
- Mostrar el resultado en consola  
- Repetir operaciones hasta que el usuario decida salir  

---

## 🧱 Estructura del Programa

### 📌 Clase Principal: `Calculadora`

#### Métodos estáticos:

- `sumar(double a, double b) : double`  
- `restar(double a, double b) : double`  
- `multiplicar(double a, double b) : double`  
- `dividir(double a, double b) : double`  

---

## 🔄 Flujo del Programa

1. Mostrar menú de opciones:
   - 1. Sumar  
   - 2. Restar  
   - 3. Multiplicar  
   - 4. Dividir  
   - 5. Salir  

2. Solicitar al usuario:
   - Dos números  
   - La operación deseada  

3. Ejecutar la operación usando `switch`

4. Mostrar el resultado

5. Repetir el proceso con un ciclo `do-while`
